# 🎁 마니또 추첨기

이 프로젝트는 **마니또를 랜덤으로 추첨**하고, 링크를 통해 각 참가자가 자기 마니또를 확인할 수 있는 웹 앱입니다.  
모바일에서도 깔끔하게 작동하며, **애니메이션 + confetti** 효과로 즐겁게 확인할 수 있습니다.

---

## 📌 사용 방법

1. **index.html 열기**  
   - 참가자 이름을 입력하고, “인원 추가” 버튼으로 이름을 더 추가할 수 있습니다.
2. **링크 생성**  
   - 각 참가자별로 링크가 생성되며, “링크 복사” 버튼으로 링크 복사 가능
3. **참가자 접속**  
   - 생성된 링크나 QR 코드로 접속
   - 접속 시 랜덤 이름 애니메이션 후 **최종 마니또 결과 + confetti** 확인
4. **마니또 확인 완료!**  
   - 링크를 다시 열어도 같은 결과 확인 가능

---

## 🎨 특징

- 모바일 최적화 UI
- 귀여운 동글동글 디자인
- 랜덤 이름 애니메이션
- confetti 효과
- 각 참가자가 자기 링크로 몰래 확인 가능
- QR 코드 및 링크 복사 기능

---

## ⚙️ 설치 및 실행

1. VS Code 또는 브라우저에서 **index.html** 열기
2. 참가자 이름 입력 → 링크 생성
3. 각 참가자가 링크 접속 → 결과 확인
